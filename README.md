# imdb-recommender

A simple movie recommender app made with Flask, BeautifulSoup and requests modules.

## Installation

```bash
pip install requirements.txt
```
(You might want to create a virtual environment beforehand)

## Usage

```bash
flask run
```
Copy and paste your ratings url
